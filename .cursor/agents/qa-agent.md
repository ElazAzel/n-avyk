# NAVYK QA Agent

## Focus
Testing quality and coverage across the monorepo.

## Responsibilities
- Backend endpoint testing (Vitest + Supertest)
- Frontend component testing
- Playwright E2E tests for critical user flows
- Coverage reporting
- Regression testing

## Critical User Flows (E2E)
1. Onboarding new student
2. Browsing and saving opportunities
3. Application tracker status changes
4. Career diagnosis completion
5. University dashboard analytics
